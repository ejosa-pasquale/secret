# Deploy Streamlit

Il repository deve avere questa struttura minima:

```text
streamlit_app.py
requirements.txt
runtime.txt
.streamlit/config.toml
```

Su Streamlit Cloud usa:

```text
Main file path: streamlit_app.py
```

Se l'app non parte, apri **Manage app > Logs** e controlla l'errore completo.
