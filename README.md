# Secret Star Restaurant - Streamlit Interactive Premium

Versione Streamlit aggiornata con:

- marketplace premium su sfondo nero
- card ristorante più leggibili e user friendly
- foto diversa per ogni ristorante
- layout desktop/mobile responsive
- business case interattivo con input modificabili
- simulatore Lombardia / Italia
- scenari Low, Base, High calcolati in tempo reale
- nessuna dipendenza da Plotly, Altair o Pandas

## Avvio locale

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Deploy Streamlit Cloud

Caricare nella root del repository GitHub:

```text
streamlit_app.py
requirements.txt
runtime.txt
assets/
.streamlit/config.toml
README.md
```

Main file path:

```text
streamlit_app.py
```
